8 - deployment & operations security

security continues after deployment and must be maintained continuously.

environment separation
Development, staging, and production environments must remain isolated.

secrets management
Secrets must never be committed to code and must be rotated regularly.

monitoring
Systems must track errors, detect anomalies, and trigger alerts for suspicious behavior.

backups
Backups must be frequent, tested, and recoverable.

patching
All systems, dependencies, and frameworks must be kept up to date.
