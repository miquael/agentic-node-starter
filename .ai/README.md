# .ai: Agentic Context Directory

Source of truth for AI agents.

## Structure

- **`AGENTS.md`**: Agent workflow, directives, and constraints.
- **`PROJECT-SETUP-STANDARD.md`**: Full rationale for each configuration decision.
- **`glossary.md`**: Project terminology.
- **`/docs/`**: Architecture, roadmap, steering, decisions, and security.
  - **`/docs/steering.md`**: Mission-critical technical and stylistic operational boundaries.
  - **`/docs/architecture.md`**: System architecture overview.
  - **`/docs/decisions.md`**: Architecture decision log.
  - **`/docs/roadmap.md`**: Project roadmap.
  - **`/docs/tasks.md`**: Current task tracking.
  - **`/docs/security/`**: OWASP-based security orientation. Load sub-docs as needed. See [security/README.md](docs/security/README.md).
  - **`/docs/specs/`**: Functional requirements and technical specifications.
